>**O objetivo principal deste projeto é **otimizar o tempo da equipe de TI**, reduzindo chamados de nível 1 (dúvidas frequentes) ao empoderar os usuários finais com guias visuais, interativos e fáceis de seguir.**

---

## 🎯 Sobre o Projeto
Este projeto nasceu da necessidade de otimizar o suporte técnico na Secretaria da Educação. Ele serve como uma base de conhecimento interativa para que secretários e diretores resolvam problemas comuns de hardware e software sem depender de chamados externos demorados.

## ✨ Funcionalidades
* **📚 Navegação Dinâmica em Árvore:** Menu lateral intuitivo que alterna os manuais na tela sem recarregar a página.
* **🖨️ Guias de Impressoras:** Tutoriais detalhados de instalação (Ex: HP Laser MFP 432 em Rede/USB) e emissão de relatórios de contador de páginas (com vídeos embutidos).
* **📹 Monitoramento de Segurança:** Manuais passo a passo do software Intelbras SIM Next (Visualização ao vivo de DVRs, Busca e Exportação de imagens).
* **🎫 Abertura de Chamados (1Doc):** Instruções padronizadas para solicitação de manutenção e refil de toner, reduzindo erros de comunicação com terceirizadas.
* **🔗 Hub de Sistemas Rapidos:** Grid interativo com acesso direto a plataformas essenciais (Betha Sistemas, Portal SED, E-mail, AnyDesk, etc).


## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica.
* **CSS3:** Estilização moderna usando Flexbox, CSS Grid, Variáveis (Custom Properties) e transições suaves.
* **JavaScript (Vanilla):** Lógica de navegação, manipulação do DOM e scroll suave (`scrollIntoView`).
* **Font Awesome:** Biblioteca de ícones (Solid/Brands) para melhorar a comunicação visual.
